# caf-gcc-starter-kit-ignite

Starter Kit for GCC-CAF-Terraform

# execute gcc starter kit using rover ignite
```bash
cd /tf/caf/ansible
rover ignite --playbook /tf/caf/ansible/gcc-starter-playbook.yml
sudo chmod -R -f 777 /tf/caf/gcc_starter
cd /tf/caf
```
